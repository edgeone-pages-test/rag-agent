数据结构
interface Message {
  messageId: string                       // msg_xxx（appendMessage 自动生成）
  role: 'user' | 'assistant' | 'system' | 'tool'
  content: any                            // string / array / object（多模态）
  createdAt: number                       // ms 时间戳
  metadata?: Record<string, any>          // 自定义（token 数、tool_call 等）
  updatedAt?: number                      // 仅 updateMessage 后存在
}
interface ConversationMeta {
  conversationId: string
  createdAt: number                       // ms 时间戳
  lastMessageAt: number                   // ms 时间戳
  messageCount: number
  metadata?: Record<string, any>          // 业务自定义（标题、用户、标签等）
}
interface ListConversationsResult {
  items: ConversationMeta[]
  nextCursor?: string                     // 下一页游标，传给 after 参数
  previousCursor?: string                 // 上一页游标，传给 before 参数
}

限制与配额
|项|默认|超限行为|
|---------|---------|---------|
|`conversation_id` 长度|≤ 256 字符|抛 `MemoryValidationError`|
|单条 `content` 大小|≤ 50MB（序列化后）|抛 `MemoryValidationError`|
